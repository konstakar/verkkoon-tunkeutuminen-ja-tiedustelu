# a) RFID suojaus

Minulla on RFID suojattu lompakko, joten pitäisi olla ihan hyvin suojattu.
En tiedä miten olen muuten suojautunut, maksukortit pidän erittäin hyvässä tallessa, mutta muut kortit kuten plussakortit/jäsenyyskotyit pidän vain samassa paikassa.

# b) APDU-komennot

googlailin ja kysyin ChatGPT:ltä niistä. Command APDU (pyyntö) CLA | INS | P1 | P2 | Lc | DATA | Le = cla = komennon tyyppi, ins(truction) = mitä tehdään, p1 ja p2 = parametrit, lc = parametrin pituus, data = varsinainen data, Le = vastaus. 

Response APDU (vastaus) = DATA = data, SW1 ja SW2 = status (onnistui/ei onnistunut)

# c) Uutinen
Hieman vanha uutinen, yli puolitoista vuotta vanha (20 elokuuta 2024). RFID-korteista löytyi piilotettu takaovi, erityisesti MIFARE Classic kortit olivat riskinalaisena, vaikutti miljooniin kortteihin maailmanlaajuisesti. Korteista löytynyt takaovi mahdollisti nopean kloonauksen. Osoitti ettei RFIS ole automaatisesti turvallinen. 

Kyseessä oli card-only attack, (pelkkä kortti riittää hyökkäykseen) ja vain muutama minuutti kortin lähellä riitti kortin kloonaamiseen. Pelkällä kortilla on huono suoja, tarvitaan vahvempi salaus ja esim. monivaiheinen tunnistus.
